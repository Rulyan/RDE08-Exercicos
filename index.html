Table

Essa tag cria uma tabela, o que usaremos é (table, tr, th e td). O table é o que vai definir que faremos uma tabela, o tr delimita
linha da tabela, o th define o titulo da tabela e o td o conteúdo da tabela.Também exitem as colspan, define quantos espaços a informação
tomara na tabela. rowspan ocupa celulas para baixo da tabela.

Exemplo de uma tabela:

O "del"(ou Elemento HTML de Texto Excluído) representa uma parte do texto que foi excluída de um documento.
Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

INS

Representa um intervalo de texto que foi adicionado a um documento.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

SUB

Define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

SUP

É basicamente um texto sobrescrito. Faz com que apareça meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

BLOCKQUOTE

Indica que o texto incluído é uma longa citação.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Q

Indica que o texto dentro da tag é uma pequena citação.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

ADDR

Representa uma abreviação e fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apenas ela.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

ADDRESS

Fornece informações de contato para seu ancestral article ou body mais próximo; no segundo caso, ele se aplica ao documento inteiro.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

CITE

Representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

BDO

É usado para substituir a direcionalidade atual do texto.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

CODE

Essa tag é usada para definir um pedaço de código de computador.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

PRE

É uma tag utilizada para representar texto pré-formatado.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

KBD

Aqui, essa tag é usada para definir a entrada do teclado.
